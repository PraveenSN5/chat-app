# UniChat - Real-time Chat Application

UniChat is a real-time chat application built with Node.js, Express, and Socket.io.

## Features

- User authentication (login/registration)
- Multiple chat rooms
- Real-time messaging
- Online user list
- Message formatting (bold, italics, links)
- Responsive design
